# Welkom bij de workshop! :)

We gaan de mogelijkheden van LLM's verder verkennen. 
Hiervoor gaan we zelf een systeem bouwen dat gebruik maakt 
van één of meerdere LLM's.

Je mag kiezen uit deze varianten:

A. [**Onderzoeker**](01-researcher.md): je bouwt zelf een AI-onderzoeker, die gebruik kan maken 
van verschillende tools.

B. [**Groepschat**](01-groupchat.md): soms heb je gewoon wat perspectief nodig! In deze versie van de opdracht bouw je een systeem dat specifiek 
bedoeld is om een prompt vanuit verschillende persona's te bekijken.

C. Als je heel graag iets anders wil proberen, overleg dit dan met de docent.

Er staan in deze folder twee losse artikelen waarin je meer kunt lezen 
over hoe je deze opdrachten uit kunt voeren. Hierbij kijken we ook waar de focus 
ligt binnen de opdracht (bijv. geheugen, retrieval-augmented generation).

## Tools

Begin met het klonen van het [startproject](https://github.com/SelinaBlijleven/Langchain4j-ollama-base). 
Deze bevat alle nodige dependencies, demo code en deze documentatie. Dat scheelt veel tijd met 
opzetten! In het startproject gebruiken we Apache Maven als package manager.

Voor deze opdracht maken we gebruik van [LangChain4J](https://docs.langchain4j.dev/), een onofficiële 
implementatie van het populaire [LangChain framework](https://www.langchain.com/) voor Python en TypeScript.

LangChain4j is nog niet helemaal volledig, waardoor sommige details iets lastiger zijn om te 
implementeren. Deze repository bevat daarom ook veel code die dat een stukje makkelijker maakt. 
De demo's laten steeds kleine beetjes zien van wat er mogelijk is en de uitdaging ligt hem in de 
onderdelen combineren om LLM's optimaal te benutten. Deze code mag je dan ook beschouwen 
als bouwblokken en je mag deze code hergebruiken en aanpassen.
